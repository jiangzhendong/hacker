# hexo的开始
## 老生常谈：
### 初始化：
``
$ hexo init hacker
``
### 第一篇文章：
``
$ hexo new "My New Post"
``
### 启动：
``
$ hexo server
``
### 编译：
``
$ hexo generate
``
### 发布：
``	
$ hexo deploy
``
